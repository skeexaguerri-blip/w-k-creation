🚀 Quick Start - Plateforme SaaS

Guide complet pour démarrer en moins de 5 minutes

⚠️ Vous lisez cette page ? C'est que vous avez les fichiers. Bravo !

1️⃣ Prérequis

Node.js & npm

Télécharger et installer depuis nodejs.org

node --version
npm --version
Un compte Supabase

Créer sur supabase.com et copier vos clés

Un compte FedaPay (optionnel pour test)

Créer sur fedapay.com pour tester les paiements

2️⃣ Installation (5 min)

Ouvrir un terminal

Naviguer vers le dossier du projet :

cd "c:\Users\Zenn\Documents\agence saas"
Installer les dépendances
npm install

Cela va installer Express, Supabase, CORS, etc.

Configurer l'environnement

Éditer le fichier .env et remplir :

SUPABASE_URL=votre_url
SUPABASE_KEY=votre_clé
FEDAPAY_SECRET_KEY=votre_clé
Démarrer le serveur
npm start

Vous devriez voir : "🚀 Serveur démarré sur http://localhost:3000"

3️⃣ Premier Test (2 min)

http://localhost:3000/index.html

8️⃣ Dépannage

❌ "Cannot GET /index.html"

→ Vous utilisez file:// au lieu de http://

✅ Solution : Utiliser http://localhost:3000/index.html

❌ "CORS policy error"

→ Les requêtes frontend→backend sont bloquées

✅ Solution : Vérifier que le serveur tourne (npm start) et que API_BASE_URL est correct

❌ "Database connection failed"

→ Les clés Supabase sont invalides

✅ Solution : Vérifier .env et copier les bonnes clés
